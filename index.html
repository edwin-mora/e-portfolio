Project 2

The Airclone web app is my full stack project created with NextJS and ReactJS on the client side and fetching data on the server side using Prisma and mongoDB. This web app is a clone of the popular airbnb website and offers users the ability to sign up using Google, Github, or email. The users also have the ability to favorite (heart) listings, rent out their own home, book a reservation on any of the site's listings, and filter out listings based on their unique category.
Technologies used: MongoDB, Node.JS, ReactJS, NextJS, TailwindCSS, NPM, Typescript
